# Prompt Templates

## 官网首页

```text
使用 home-ui-design-skill 为 [audience] 创建首页。包含紧凑顶部导航、平静 hero、模块化内容卡片、可选侧栏或次级组件、清晰 CTA、响应式行为和可访问性说明。缺少内容时使用中性占位文案。
```

## Landing Page

```text
使用 home-ui-design-skill 为 [offer] 创建落地页。包含 hero、功能卡片、证明 section、产品或服务细节、CTA 面板、页脚、design tokens、交互状态和移动端优先布局规则。
```

## 产品介绍区

```text
使用 home-ui-design-skill 为 [product category] 生成产品介绍区。包含 section 标题、短收益文案、功能列表、媒体块、CTA、token 用法、响应式行为和可访问性要求。
```

## 移动端页面

```text
使用 home-ui-design-skill 设计 [task] 的移动端页面。优先考虑紧凑顶部导航、图标控件、全宽卡片、可读标题、触控目标、浮层、reduced motion 和中性占位内容。
```

## 组件规范

```text
使用 home-ui-design-skill 为 [components] 创建组件规范。每个组件包含使用场景、视觉结构、token 依赖、状态、响应式行为、可访问性、实现提示、禁止模式和中性示例。
```

## Design Tokens

```text
使用 home-ui-design-skill 为 [interface type] 生成语义化 design tokens。包含 color、typography、spacing、radius、shadow、border、motion、breakpoint、z-index、opacity、blur、container 和 grid tokens。每个 token 包含 role、usage、confidence、modifiable 和 fallback。
```

## React 实现

```text
使用 home-ui-design-skill 为 [page or component] 创建 React 实现。定义 token CSS variables、语义组件、状态 props、响应式行为、键盘支持和 reduced-motion 处理。
```

## Vue 实现

```text
使用 home-ui-design-skill 为 [page or component] 创建 Vue 实现。定义 token CSS variables、组件 props、状态 class bindings、响应式行为、键盘支持和 reduced-motion 处理。
```

## HTML CSS 实现

```text
使用 home-ui-design-skill 为 [page or component] 创建 HTML/CSS 实现。使用语义 landmarks、CSS variables、圆角卡片表面、紧凑控件、响应式 media queries、可见 focus 和 reduced-motion rules。
```

## Tailwind 实现

```text
使用 home-ui-design-skill 为 [page or component] 创建 Tailwind 实现计划。将 design tokens 映射到 Tailwind theme values，必要时使用语义组件类，并包含响应式 variants、状态和可访问性要求。
```

## 色彩风格替换

```text
使用 home-ui-design-skill 将主 hue 替换为 [hue or color direction]。保持卡片几何、间距、层级和交互行为稳定。重新检查对比度、focus 可见性、暗色模式和选中状态。
```

## 组件风格替换

```text
使用 home-ui-design-skill 将组件风格调整为 [technical, premium, minimal, corporate, or product-led]。保持 tokens 语义化，保留可访问性，并列出改变的 tokens。
```

## 质量检查

```text
使用 home-ui-design-skill 校验该 UI 输出。检查 token 使用、布局语法、组件配方、状态、响应式行为、可访问性、中性内容、私有资产规避和发布清洁度。
```
