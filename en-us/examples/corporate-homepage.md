# Corporate Homepage Example

## Input Request

Create a homepage for a professional services team. Use a soft, trustworthy, modular system with restrained visuals.

## Skill Rules Used

- Corporate adaptation with calm spacing.
- Top navigation card.
- Split content hero.
- Service cards.
- Testimonial or proof section.
- Footer utility links.

## Generation Strategy

- Use a restrained cool accent and larger whitespace.
- Use service cards with concise descriptions.
- Place proof after capability sections.
- Keep CTA copy direct and low-pressure.

## Structure Draft

1. Navigation.
2. Split hero with primary CTA.
3. Service overview cards.
4. Process section.
5. Proof cards or testimonial.
6. CTA strip.
7. Footer.

## Token Use

- `color.brand.primary` for CTA and active indicators.
- `color.text.secondary` for supporting copy and metadata.
- `grid.feature.cards` for services.
- `shadow.none` for static cards.
- `shadow.panel.float` only for dropdowns.

## Responsive Notes

- Desktop hero can use two columns.
- Tablet hero becomes balanced stacked content if needed.
- Mobile hero uses text first and full-width actions.
- Service cards stack on mobile.

## Accessibility Notes

- Navigation uses semantic links.
- CTA buttons have accessible labels.
- Testimonials avoid auto-rotation.
- Color is not the only marker for recommended content.

## Quality Check

- Professional tone.
- No crowded cards.
- Clear section hierarchy.
- Strict contrast and focus checks.
- Replaceable placeholder copy.
